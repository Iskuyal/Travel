# 住宿推荐

> **行程**：7天6晚 · **全程住潮州古城**（无需搬迁）
>
> **⚠️ 硬性筛选条件**：
> 1. **提供厨房或可自己开火做饭**（能自己准备早餐）
> 2. **预算 ≤ 500元/晚/间**（7月旺季参考价，整栋8人需×房间数）
> 3. **位置在潮州古城步行1公里范围内**
> 4. **整套出租，非合住**

> 📢 **重要提示**：7月旺季古城核心区「整栋别墅+厨房齐全+8人+500元以内」组合极为稀缺。以下推荐按"性价比最优"排序，部分房源整栋价格可能超过¥500/晚，但人均仍可控。建议尽早预订。

## 如何找到符合条件的住宿

| 平台 | 优势 | 筛选方法 |
|------|------|----------|
| **携程** (ctrip.com) | 评价体系最完善 | 搜索"潮州古城" → 类型选"民宿/公寓" → 设施选"厨房" → 价格≤500 |
| **途家** (tujia.com) | 公寓类房源最多，厨房标配 | 搜索"潮州牌坊街" → 筛选"整租" → "可做饭" → 价格≤500 |
| **美团民宿** (minsu.meituan.com) | 常有最低价 | 搜索"潮州古城" → 筛选"可做饭" → 价格排序 |
| **Trip.com** | 港台游客评价多 | 搜索"Chaozhou" → Amenity选"Kitchen" → 价格筛选 |
| **小红书** | 真实体验分享 | 搜索"潮州古城 民宿 可做饭 带厨房"看实拍 |

> 📞 **预订前必问房东**：①可以做饭吗？电磁炉还是明火？②有冰箱吗？③锅具/餐具齐全吗？④附近有菜市场吗？⑤有额外清洁费吗？⑥停车怎么收费？
