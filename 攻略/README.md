## ⚡ 快速参考

### 🚗 深圳 → 潮州（9人·2辆车）
- 路线：沈海高速 (G15)，约 350-360 公里
- 耗时：约 3.5-4.5 小时
- 过路费约 ¥170-190/车（单程）

### 🗺️ 行程概览（7天6晚）

| 天数 | 主题 | 核心景点 |
|------|------|----------|
| Day 1 | 古城初探 | 西湖公园·牌坊街·西马路·灯光秀 |
| Day 2 | 文化精华 | 开元寺·己略黄公祠·许驸马府·广济桥·韩文公祠 |
| Day 3 | 城外东线 | 淡浮院·潮人公园 |
| Day 4 | 古寨风情 | 龙湖古寨·泰佛殿·载阳茶馆 |
| Day 5 | 地质奇观 | 青岚怪臼谷（全天） |
| Day 6 | 城市漫步 | 慧如公园·青龙古庙·北阁佛灯·收官采购 |
| Day 7 | 满载而归 | 返程深圳 |

### ⚠️ 关键提醒
- **周一**：韩文公祠、己略黄公祠、许驸马府、潮州市博物馆闭馆（行程已安排周四参观）
- **灯光秀**：周一至周四 20:00一场，周五至周日 20:00+21:30两场（**周一有秀！**）
- **广济桥夜游**：周五六日 19:30-21:30，**推广期¥99/人（原价¥169）**，**限西桥100人/晚**，需"一票潮州"公众号**提前36小时**预约
- **大团队**：提前建微信群共享位置，餐厅务必提前订桌
- **淡浮院**：网上购票需提前1小时，比现场买便宜40%+
- **饮食禁忌**：同行人员不食用生鲜/刺身/生腌/鱼生，所有餐饮推荐已避开
